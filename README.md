# squareWare_startCooking

This is a front end mobile challenge.

How to install and run this project:-
1. clone this project in your system.
2. in cmd or powershell go to this project directory.
3. run command npm install, this command will install all the dependency required for the project.
4. after installing all the dependency.
5. run command npm start, this command initiallize the project. 
6. You can run this project in expo mobile app, android emulator, ios emulator and on web also.
